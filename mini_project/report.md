## 3. 통계적 가설 검정 (t-test) 및 p-value 해석
### 3-1. 연봉 격차 검증 결과
- **AI 사용 그룹 평균**: `$75,031.31` / **미사용 그룹 평균**: `$84,726.37`
- **t-통계량**: `-11.4657` / **p-value**: `2.5099e-30` (`0.0000`)
- **해석**: p-value(0.0000) < 0.05이므로 연봉 차이가 통계적으로 유의미합니다. 👉 결론: 실제 데이터 분석 결과, AI 도구를 사용하지 않는 개발자의 평균 연봉이 유의미하게 더 높습니다!

### 3-2. 직무 만족도 격차 검증 결과
- **AI 사용 그룹 만족도**: `6.97점` / **미사용 그룹 만족도**: `6.89점`
- **t-통계량**: `3.5240` / **p-value**: `4.2612e-04` (`0.0004`)
- **해석**: p-value(0.0004) < 0.05이므로 직무 만족도 차이가 통계적으로 유의미(p < 0.05)합니다. 👉 결론: AI 도구를 사용하는 그룹이 미사용 그룹보다 직무 만족도(JobSat)가 통계적으로 유의미하게 높습니다!
