# CRISP-DM 기반 글로벌 vs 대한민국 개발자 연봉 예측 및 통계 분석
> Stack Overflow Developer Survey 2024 데이터를 활용해 국가/경력/학력에 따른 개발자 보상 격차를 통계적으로 검증하고 적정 연봉을 예측하는 End-to-End 분석 파이프라인 및 배포 프로토타입 프로젝트입니다.

---

## 사용 기술 및 환경
* **언어**: Python (v3.10+)
* **데이터 전처리 & 고속 로딩**: `Pandas`, `Polars` (Lazy Evaluation 비교 분석)
* **통계적 가설 검정**: `SciPy` (Levene, Independent t-test, Pearson Correlation, One-way ANOVA), `Statsmodels` (OLS Regression)
* **머신러닝 파이프라인**: `Scikit-Learn` (ColumnTransformer, Pipeline, SimpleImputer, StandardScaler, OneHotEncoder, Ridge Regression)
* **모델 영속화**: `joblib`
* **시각화**: `Seaborn`, `Matplotlib` (정적 차트), `Plotly` (반응형 동적 HTML 차트)

## 프로젝트 기획 및 의사결정 과정 (Discussion & Decision Process)
본 프로젝트는 특정 상용 서비스 기능 구현보다는 **현업 데이터에 기반한 의혹 해결 및 가설 검증용 보고서 도출**을 주 목표로 삼았기에, 팀원들 간의 정밀한 의논과 의사결정 과정을 통해 분석을 설계하였습니다.

### 1단계. 최초의 고민과 과제 분석
* 과제를 처음 인계받았을 때 제공된 3가지 후보 데이터셋(NYC Taxi, Stack Overflow, Adult Census) 중 어떠한 비즈니스 문제에 초점을 맞출지 고민이 있었습니다.
* **NYC Taxi 데이터**는 단순 대용량 병렬 연산 속도 측정에는 적합하지만 비즈니스 인사이트 도출 범위가 좁았고, **Adult Census 데이터**는 연봉 데이터가 50K 초과 여부로 범주화되어 있어 회귀 예측의 다양성을 살리기 힘든 한계가 있었습니다.

### 2단계. 데이터셋 최종 선정 (관심사와 분석 방법의 다양성)
* 팀원 모두가 공통적으로 큰 관심을 가진 **'IT 개발자의 실제 보상(Salary)과 커리어의 연관성'**에 부합하는 **Stack Overflow Survey 2024** 데이터를 분석 대상으로 선정하였습니다.
* 해당 데이터는 수치형(경력 연차)과 고차원 범주형(180개 이상의 국가, 다양한 학력 수준, 수십 종의 기술 스택) 변수가 조화롭게 존재하여, t-test부터 분산분석(ANOVA), 다중회귀분석(OLS), 파이프라인 머신러닝 예측까지 **분석 기법의 다양성을 최대로 실험**할 수 있다는 점이 결정적이었습니다.

### 3단계. 원본 데이터 분석 및 구체적 분석 방향 수립
* 약 155MB 규모의 대용량 설문 원본 데이터를 로드해본 결과, 결측치가 60% 이상 존재하고 극단치(연봉 $10,000 이하 또는 $500,000 이상)가 산재해 있음을 파악했습니다. 또한, 국가명에 `Republic of Korea`와 `South Korea`가 혼재되어 있었습니다.
* 데이터 정제 기준을 정한 후, **"한국 개발자의 보상 수준은 과연 글로벌 시장 평균과 통계적으로 유의미한 차이가 있는가?"**라는 메인 연구 가설을 수립했습니다.
* 가설을 정량적으로 증명하기 위해 `is_korea` 파생변수를 설계하고, t-검정과 OLS 모델로 영향력을 해석한 뒤, 기업 인사 담당자가 채용 시 활용할 수 있는 **'실시간 다차원 연봉 예측 CLI 시뮬레이션 배포'**를 최종 목표로 고정하였습니다.
